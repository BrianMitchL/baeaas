BAEaaSBAE as a Service

A RESTful API for BAEifying your life.

Operations

Path Description
/version Will return the current fersion of BAEaaS
/operations Will return a JSON list of operations with names and fields
/bae/:text Will BAEify the given string, e.g. /bae/This%20way%20to%20bae will return 'This bae to bae'
